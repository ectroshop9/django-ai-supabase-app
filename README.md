# مشروع SerialCo - متجر المنتجات الرقمية (Django API)

هذا المشروع هو واجهة برمجية (API) خلفية مبنية على Django، مصممة لإدارة بيع المنتجات الرقمية عبر نظام الكوينز.

## ⚙️ البنية التحتية

* **قاعدة البيانات:** PostgreSQL عبر Supabase
* **التخزين المؤقت:** Redis
* **توزيع الملفات:** Cloudflare R2 + CDN
* **حماية التنزيل:** Cloudflare Workers (لتوليد التوكنات المؤقتة)
